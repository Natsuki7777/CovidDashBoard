# プロジェクトの概要

コロナダッシュボードは、企業の方などに、
「自分達が管理画面などを作れる(実績がある)」ことを示すために作ります。
ささっと Ver.1 を作りましょう

# 開発に参加するまでの流れ

以下の手順を踏んで開発に参加しましょう。

## 1. `git clone`

以下のコマンドを実行してクローンします。

```shell
git clone https://github.com/dokosore/coviddashboard.git
```

## 2. `npm install`

npm でパッケージをインストールします

```shell
#パッケージをインストール
npm install
```

## 3. `npm run dev`

`npm run dev`でローカルサーバーを立ち上げます。

```shell
npm run dev
```
